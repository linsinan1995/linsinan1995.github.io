Hugo Tutorial

1. Quick start

install hugo

sudo apt install hugo

2.1. Create a new site

hugo new site myBlog
cd myBlog

2.2.Add a theme

git init
git submodule add https://github.com/zwbetz-gh/minimal-bootstrap-hugo-theme.git themes/minimal-bootstrap-hugo-theme
rm config.toml
cp themes/minimal-bootstrap-hugo-theme/exampleSite/config.toml config.toml

2.3.Post your first article

command hugo new posts/xxx.md will generate a xxx.md file in the directory content/posts/

hugo new posts/my-first-post.md

2.4.Add a tag for your posts

add tags: ["tutorial"] in between two --- at the top of your .md file.

5.Deploy on Github

Now, let’s test it in the browser! paramater -D will generate static pages based on your edits.

hugo server -D

Drafts do not get deployed; once you finish a post, update the header of the post to say draft: false.
